# DVLD Project

A desktop application for managing the Driving and Vehicle License Department (DVLD), developed using C# WinForms, SQL Server, and a Three-Tier Architecture.

This project is built by following the Programming Advices course while applying clean architecture and object-oriented programming principles.

---

## Features

- People Management
- Users Management
- Application Types Management
- Test Types Management
- Local Driving License Applications
- Driving License Classes
- Vision Test Management
- Written Test Management
- Street Test Management
- Drivers Management
- Licenses Management
- Detained Licenses
- International Licenses
- User Authentication
- Permissions & Access Control

## Architecture

The project follows the Three-Tier Architecture:

Presentation Layer
- User Interface
- Forms
- User Controls

Business Layer
- Business Logic
- Validation
- Rules
- Services

Data Access Layer
- SQL Server
- ADO.NET
- CRUD Operations
